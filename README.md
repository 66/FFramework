## 工程介绍  

此工程热更模块基于 HybridCLR + Addressable 

1. https://github.com/focus-creative-games/HybridCLR wolong
2. https://docs.unity3d.com/Packages/com.unity.addressables@1.20/manual/index.html AddressablesAssetPlugins

同时支持资源和逻辑热更的基础工程
